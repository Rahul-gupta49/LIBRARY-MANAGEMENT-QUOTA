# LIBRARY-MANAGEMENT-QUOTA

A Library Management System for managing books, members, and borrowing quotas.

## General Inquiries / FAQ

Have questions that aren't about code? Here are answers to commonly asked questions about the Library Management System.

### What is this project?

This is a Library Management System designed to help libraries manage their book inventory, track member borrowing activity, and enforce borrowing quotas.

### What features does the system support?

- **Book Management** – Add, update, remove, and search for books in the catalog.
- **Member Management** – Register members and manage their accounts.
- **Borrowing & Returns** – Track which books are borrowed and returned.
- **Quota Enforcement** – Set and enforce limits on how many books a member can borrow at one time.

### Who is this project for?

This project is intended for small to medium-sized libraries, educational institutions, or anyone who needs a simple system to manage book lending.
